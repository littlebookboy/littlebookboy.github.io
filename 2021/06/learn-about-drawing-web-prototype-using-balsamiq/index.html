工具

https://balsamiq.com/

純粹興趣練習畫看看 Web Prototype，網路上找到的推薦工具很多，就挑了個相對順眼 + 操作簡單 + 免費試用 + 有桌面軟體 + 有官方指引的 這套 來玩玩。

balsamiq 的 MAC 桌面版安裝好之後，圖示長這樣

直接上影片

最後

這套使用上很好理解 (練習影片大概花不到兩小時做的)，而且很多別人做好的 layout，可以直接匯入使用修改，大大節省了重新刻輪子的時間；

目前官方售價桌面軟體買斷是一人 89 鎂，不知道月租除了可以月付少少 - 9 鎂 (不用一次買斷) 優點，以及限制最多創建兩個專案 (嗯好像有點少) 之外，有無功能上的差異；

另外還有個付費方案是能整合到 Google 雲端、Atlassian Confluence，或是 Jira ；

有任何問題或想法，歡迎留言交流，如果寫的內容有錯誤的地方，希望能不吝指點，感謝。
